Serenite - Journal de Version

4.3.0

12-02-2024
Nouveau
  • L'utilisateur peut maintenant voir sur quelle version il est connectée dans le menu latéral gauche.
  • L'utilisateur administrateur peut maintenant rechercher et afficher les enquêtes par arbitre.
Technique
  • [RGPD] Le mécanisme complet de cryptage des données personnelles a été mis à jour.
  • [RGPD] Le mécanisme complet de suivi des modifications de données personnelles a été mis à jour.
  • [RGPD] Un service automatisé de suppression et d'anonymisation des comptes utilisateurs est disponible.
  • Les utilisateurs administrateurs n'ont plus besoin d'avoir une licence pour être créé.
  • Une liaison technique est maintenant effectué entre les clubs et leurs bassins territoriaux.
Correction
  • Ajout d'une gestion des erreurs à la connexion de l'utilisateur (mot de passe incorrect, plateforme non disponible, etc...).

4.2.0

25-01-2024
Nouveau
  • Ajout d'un menu latéral permettant de se déconnecter et d'accéder à la création de rencontre.
  • L'utilisateur peut maintenant créer une rencontre dans la nouvelle interface depuis le site web.
  • L'utilisateur peut maintenant créer une rencontre dans la nouvelle interface depuis son smartphone.
  • Le champ de mot de passe est maintenant caché lors de la connexion.
Correction
  • Le commentaire est limité à 150 caractères.
  • Les équipes affichées dans le résumé de l'enquête sont cohérentes.

  • L'utilisateur peut maintenant effectuer la procédure de mise à jour du mot de passe dans la nouvelle interface .
