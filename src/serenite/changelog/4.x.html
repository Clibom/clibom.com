4.4.0

05-05-2024
Nouveau
  • L'administrateur peut maintenant supprimer les rencontres d'un utilisateur.
  • L'administrateur peut créer les licences manquantes ou non importées automatiquement.
  • L'administrateur peut consulter une fiche d'information générale concernant un utilisateur.
  • L'administrateur/formateur peut maintenant voir le détail des enquêtes par utilisateur.
  • L'administrateur/statisticien peut maintenant voir les statistiques des clubs.
  • L'administrateur/statisticien peut maintenant voir le détails des enquêtes par équipes.
Technique
  • Ajout d'une date de création de l'enquête en base de données.
  • Création d'un portail support dédié : ICI.
  • Création d'une base documentaire dédiée : ICI.
Correction
  • [Création de rencontre] Gestion du code de rencontre au format normé.
  • [Création de rencontre] La création de rencontre possède maintenant un parcours unique qui permet de recommencer.
  • [Création de rencontre] Les rencontres créées possédait le même recevant et le même visiteur.
  • [Apple] Les étoiles de notation ont été remplacés par une barre de selection propre aux produits Apple.

4.3.0

12-02-2024
Nouveau
  • L'utilisateur peut maintenant voir sur quelle version il est connecté dans le menu latéral gauche.
  • L'utilisateur administrateur peut maintenant rechercher et afficher les enquêtes par arbitre.
Technique
  • [RGPD] Le mécanisme complet de cryptage des données personnelles a été mis à jour.
  • [RGPD] Le mécanisme complet de suivi des modifications de données personnelles a été mis à jour.
  • [RGPD] Un service automatisé de suppression et d'anonymisation des comptes utilisateurs est disponible.
  • Les utilisateurs administrateurs n'ont plus besoin d'avoir une licence pour être créé.
  • Une liaison technique est maintenant effectué entre les clubs et leurs bassins territoriaux.
Correction
  • Ajout d'une gestion des erreurs à la connexion de l'utilisateur (mot de passe incorrect, plateforme non disponible, etc...).
